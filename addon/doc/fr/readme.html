NVoice - Assistant Vocal IA pour NVDA

Auteur: Melhym Quemel

Version: 2.0.0

1. À propos de NVoice

NVoice est une extension qui transforme NVDA en un assistant conversationnel IA à très faible latence. Avec lui, vous pouvez dialoguer avec les modèles d'IA les plus avancés au monde en utilisant simplement votre voix. NVoice n'utilise pas de bibliothèques tierces lourdes; il diffuse le texte IA directement vers la voix native de NVDA en temps réel, compatible avec NVDA 32 bits et 64 bits.

2. Raccourcis Clavier (Commandes Globales)

NVoice a été conçu pour être déclenché rapidement sans interrompre votre flux de travail:

NVDA + Z
Démarrer/Arrêter Conversation: Appuyez pour que NVoice commence à écouter votre microphone. NVDA annoncera "Écoute...". Parlez ce que vous voulez et appuyez à nouveau sur le raccourci pour envoyer l'audio.
NVDA + Shift + Z
Envoyer le Contexte de l'Écran: Fonctionne comme la commande précédente, mais capture le texte où se trouve le curseur NVDA et l'ajoute à votre question.
NVDA + Alt + Z
Effacer l'Historique: Réinitialise la mémoire de contexte de la conversation actuelle. Cette commande n'efface pas l'historique enregistré dans un fichier.

3. Configuration des Fournisseurs IA

Allez dans NVDA + N > Préférences > Paramètres > NVoice pour configurer:

Groq (Recommandé pour STT)

Groq utilise une architecture de traitement ultra-rapide (LPU). Recommandé pour la transcription vocale (STT) avec Whisper-Large.

  • Clé API: Groq Console
  • Coût: Quota gratuit extrêmement généreux.

Google Gemini

Intelligence artificielle de Google. Excellent pour des réponses fluides et des analyses détaillées.

  • Clé API: Google AI Studio

OpenAI (ChatGPT)

Offre les puissants modèles GPT-5 et supérieurs.

  • Clé API: OpenAI Platform

OpenRouter (Multi-Modèles)

Avec une seule clé, accédez à des centaines de modèles (Claude, Mistral, Llama, etc.).

  • Clé API: OpenRouter Keys

4. Vie Privée Totale: IA Locale avec Ollama

Pour une vie privée absolue ou un travail hors ligne, NVoice s'intègre à Ollama. Configurez à l'adresse par défaut http://localhost:11434.

Modèles Recommandés pour Ordinateurs Modestes:

  • qwen2:0.5b - Extrêmement petit et rapide.
  • llama3.2:1b - Super léger, excellent pour les textes rapides.
  • phi3:mini - Haute capacité de raisonnement.

Pour télécharger: ollama pull llama3.2:1b

5. Tutoriel: Installer Whisper et Ollama (Windows)

Étape 1: Installer Python et FFmpeg

Ouvrez l'Invite de Commandes en tant qu'Administrateur et exécutez:

winget install Python.Python.3.11

winget install Gyan.FFmpeg

Redémarrez votre ordinateur après avoir installé FFmpeg.

Étape 2: Installer Whisper

pip install -U openai-whisper

Étape 3: Installer Ollama

winget install Ollama.Ollama

Étape 4: Télécharger les Modèles

ollama pull qwen2.5:0.5b

ollama pull llama3.2:1b
