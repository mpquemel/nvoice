NVoice - KI-Sprachassistent für NVDA

Autor: Melhym Quemel

Version: 2.0.0

1. Über NVoice

NVoice ist eine Erweiterung, die NVDA in einen KI-gestützten Konversationsassistenten mit sehr geringer Latenz verwandelt. Damit können Sie mit den fortschrittlichsten KI-Modellen der Welt allein durch Ihre Stimme dialogisieren. NVoice verwendet keine schweren Bibliotheken von Drittanbietern; es streamt KI-Text direkt in Echtzeit in die Native Stimme von NVDA, kompatibel mit NVDA 32-Bit und 64-Bit.

2. Tastenkürzel (Globale Befehle)

NVoice wurde entwickelt, um schnell ausgelöst zu werden, ohne Ihren Arbeitsablauf zu unterbrechen:

NVDA + Z
Gespräch starten/stoppen: Drücken Sie, damit NVoice beginnt, auf Ihr Mikrofon zu hören. NVDA kündigt "Hören..." an. Sprechen Sie, was Sie möchten, und drücken Sie die Tastenkombination erneut, um die Audio zu senden.
NVDA + Shift + Z
Screenkontext senden: Funktioniert wie der vorherige Befehl, erfasst jedoch den Text, wo sich der NVDA-Cursor befindet, und hängt ihn an Ihre Frage an.
NVDA + Alt + Z
Verlauf löschen: Setzt den Kontextspeicher der aktuellen Unterhaltung zurück. Dieser Befehl löscht nicht den in einer Datei gespeicherten Verlauf.

3. KI-Anbieter konfigurieren

Gehen Sie zu NVDA + N > Einstellungen > NVoice zum Konfigurieren:

Groq (Für STT empfohlen)

Groq verwendet ultraschnelle Verarbeitungsarchitektur (LPU). Empfohlen für Spracherkennung (STT) mit Whisper-Large.

  • API-Schlüssel: Groq Console
  • Kosten: Extrem großzügiges kostenloses Kontingent.

Google Gemini

Künstliche Intelligenz von Google. Ausgezeichnet für flüssige Antworten und detaillierte Analysen.

  • API-Schlüssel: Google AI Studio

OpenAI (ChatGPT)

Bietet leistungsstarke GPT-5-Modelle und höher.

  • API-Schlüssel: OpenAI Platform

OpenRouter (Multi-Modelle)

Mit einem einzigen Schlüssel haben Sie Zugriff auf Hunderte von Modellen (Claude, Mistral, Llama usw.).

  • API-Schlüssel: OpenRouter Keys

4. Totale Privatsphäre: Lokale KI mit Ollama

Für absolute Privatsphäre oder Offline-Arbeit integriert sich NVoice mit Ollama. Konfigurieren Sie unter der Standardadresse http://localhost:11434.

Empfohlene Modelle für bescheidene Computer:

  • qwen2:0.5b - Extrem klein und schnell.
  • llama3.2:1b - Super leichtgewichtig, hervorragend für schnelle Texte.
  • phi3:mini - Hohe Denkfähigkeit.

Zum Herunterladen: ollama pull llama3.2:1b

5. Tutorial: Whisper und Ollama installieren (Windows)

Schritt 1: Python und FFmpeg installieren

Öffnen Sie die Eingabeaufforderung als Administrator und führen Sie aus:

winget install Python.Python.3.11

winget install Gyan.FFmpeg

Starten Sie Ihren Computer nach der Installation von FFmpeg neu.

Schritt 2: Whisper installieren

pip install -U openai-whisper

Schritt 3: Ollama installieren

winget install Ollama.Ollama

Schritt 4: Modelle herunterladen

ollama pull qwen2.5:0.5b

ollama pull llama3.2:1b
