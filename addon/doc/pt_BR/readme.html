NVoice - Assistente de Voz com IA para NVDA

Autor: Melhym Quemel

Versão: 2.0.0

1. Sobre o NVoice

O NVoice é um complemento que transforma o seu NVDA em um assistente cognitivo conversacional de baixíssima latência. Com ele, você pode dialogar com os modelos de Inteligência Artificial mais avançados do mundo usando a sua própria voz. O NVoice não utiliza bibliotecas pesadas de terceiros; ele faz o processamento de streaming enviando o texto da IA diretamente para a voz nativa do seu NVDA em tempo real, compatível com NVDA de 32 bit e 64 bit.

2. Atalhos de Teclado (Comandos Globais)

O NVoice foi desenhado para ser acionado rapidamente, sem interromper o seu fluxo de trabalho:

NVDA + Z
Iniciar/Parar Conversa: Pressione para que o NVoice comece a ouvir o seu microfone. O NVDA anunciará "Ouvindo...". Fale o que desejar e pressione o atalho novamente para enviar o áudio.
NVDA + Shift + Z
Enviar Contexto da Tela: Funciona como o comando anterior, mas captura o texto onde o cursor do NVDA está focado e anexa à sua pergunta.
NVDA + Alt + Z
Limpar Histórico: Zera a memória de contexto da conversa atual. Este comando não apaga o histórico salvo em arquivo.

3. Configurando Provedores de IA

Acesse NVDA + N > Preferências > Configurações > NVoice para configurar:

Groq (Recomendado para STT)

O Groq utiliza arquitetura de processamento ultrarrápida (LPU). Recomendado para transcrição de voz (STT) usando Whisper-Large.

  • API Key: Groq Console
  • Custo: Franquia gratuita extremamente generosa.

Google Gemini

Inteligência artificial do Google. Excelente para respostas fluídas e análises detalhadas.

  • API Key: Google AI Studio

OpenAI (ChatGPT)

Oferece os poderosos modelos GPT-5 e superiores.

  • API Key: OpenAI Platform

OpenRouter (Multi-Modelos)

Com uma única chave, acesse centenas de modelos (Claude, Mistral, Llama, etc.).

  • API Key: OpenRouter Keys

4. Privacidade Total: Rodando IA Local com Ollama

Para privacidade absoluta ou trabalho offline, o NVoice se integra ao Ollama. Configure no endereço padrão http://localhost:11434.

Modelos Recomendados para Computadores Modestos:

  • qwen2:0.5b - Extremamente pequeno e rápido.
  • llama3.2:1b - Super enxuto, excelente para textos rápidos.
  • phi3:mini - Alta capacidade de raciocínio.

Para baixar: ollama pull llama3.2:1b

5. Tutorial: Instalando Whisper e Ollama (Windows)

Passo 1: Instalar Python e FFmpeg

Abra o Prompt de Comando como Administrador e execute:

winget install Python.Python.3.11

winget install Gyan.FFmpeg

Reinicie o computador após instalar o FFmpeg.

Passo 2: Instalar Whisper

pip install -U openai-whisper

Passo 3: Instalar Ollama

winget install Ollama.Ollama

Passo 4: Baixar Modelos

ollama pull qwen2.5:0.5b

ollama pull llama3.2:1b
