# NVoice - Assistente de Voz com IA para NVDA ## Sobre NVoice é um add-on que permite conversar com IAs usando sua voz. Pressione **NVDA+Z** para ativar, fale, e receba respostas por áudio. ## Recursos - **Suporte a múltiplos providers:** - Ollama (local, sem internet) - Groq (STT com Whisper + chat com créditos free) - Google Gemini - OpenAI GPT-4 - OpenRouter (multi-modelo) - **STT (Speech-to-Text):** - Groq Whisper-large-v3-turbo (cloud, free tier generoso) - Whisper local (fallback, requer faster-whisper instalado) - **TTS (Text-to-Speech):** - Microsoft Edge TTS (vozes neurais) - Antonio Neural (padrão, +40% rate) - Francisca Neural, Thalita Multilingual - **Configurações completas:** - Seleção de provider e modelo - Validação de API key em tempo real - Lista de modelos disponíveis - Ajuste de voz, rate, volume - Comportamento (compartilhar contexto, histórico, etc.) ## Instalação 1. Baixe o arquivo `.nvda-addon` 2. No NVDA: `Ferramentas → Gerenciar complementos → Instalar` 3. Selecione o arquivo e confirme 4. Reinicie o NVDA ## Uso ### Ativar NVoice - Pressione **NVDA+Z** para começar a falar - Fale seu comando/pergunta - Pressione **NVDA+Z** novamente para parar - O NVoice processa e responde por voz ### Configurar 1. NVDA → `Preferências → NVoice` 2. Selecione o provider (Ollama, Groq, Gemini, etc.) 3. Insira API key (se necessário) e clique em "Testar" 4. Clique em "Buscar Modelos" para listar disponíveis 5. Selecione o modelo desejado 6. Configure voz TTS (Antonio, Francisca, etc.) 7. Ajuste rate e volume 8. Salve ## Providers ### Ollama (Local) - **Endpoint:** `http://localhost:11434` (padrão) - **API Key:** Não necessária - **Modelos:** Qualquer modelo instalado localmente - **Vantagens:** Privacidade total, sem internet, sem custo - **Requisitos:** Ollama instalado e rodando ### Groq - **API Key:** Necessária (https://console.groq.com) - **STT:** whisper-large-v3-turbo (free, generoso) - **Chat:** llama-3.3-70b-versatile, mixtral, etc. (free tier) - **Vantagens:** Muito rápido, créditos free generosos ### Google Gemini - **API Key:** Necessária (https://makersuite.google.com) - **Modelos:** gemini-2.0-flash, gemini-pro, etc. - **Vantagens:** Multimodal (aceita imagens), bom para contexto visual ### OpenAI GPT-4 - **API Key:** Necessária (https://platform.openai.com) - **Modelos:** gpt-4o, gpt-4-turbo, gpt-3.5-turbo - **Vantagens:** Modelo mais capaz, GPT-4 Vision para imagens ### OpenRouter - **API Key:** Necessária (https://openrouter.ai) - **Modelos:** Centenas (Llama, Mistral, Claude, etc.) - **Vantagens:** Acesso a múltiplos modelos com uma chave, pay-per-use ## Atalhos de Teclado | Atalho | Ação | |--------|------| | NVDA+Z | Ativar/desativar escuta | | NVDA+Shift+Z | Abrir configurações | ## Requisitos - NVDA 2025.3.3 ou superior - Python 3.11+ (incluído no NVDA 2026+) - Para STT local: `faster-whisper` (opcional, fallback) - Para Ollama: Ollama instalado em http://localhost:11434 - Para providers cloud: API key válida ## Estrutura do Projeto ``` nvoice-addon/ ├── addon/ │ ├── globalPlugins/ │ │ └── nvoice/ │ │ ├── __init__.py # Plugin principal │ │ ├── nvoice_config.py # Gerenciador de config │ │ ├── nvoice_core.py # Processamento (STT/IA/TTS) │ │ └── nvoice_gui.py # Interface de configurações │ └── manifest.ini ├── buildVars.py └── readme.html ``` ## Desenvolvimento ### Build ```bash scons ``` ### Install (dev) ```bash # Copiar para pasta do NVDA cp nvoice-1.0.0.nvda-addon "%APPDATA%\nvda\addons\nvoice" ``` ## Roadmap (v2.0) -
