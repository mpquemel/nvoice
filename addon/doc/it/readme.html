NVoice - Assistente Vocale IA per NVDA

Autore: Melhym Quemel

Versione: 2.0.0

1. Informazioni su NVoice

NVoice è un componente aggiuntivo che trasforma NVDA in un assistente conversazionale IA a bassissima latenza. Con esso, puoi dialogare con i modelli di IA più avanzati al mondo usando solo la tua voce. NVoice non utilizza librerie esterne pesanti; trasmette il testo IA direttamente alla voce nativa di NVDA in tempo reale, compatibile con NVDA a 32 bit e 64 bit.

2. Scorciatoie da Tastiera (Comandi Globali)

NVoice è progettato per essere attivato rapidamente senza interrompere il flusso di lavoro:

NVDA + Z
Avvia/Interrompi Conversazione: Premi per far iniziare NVoice ad ascoltare il tuo microfono. NVDA annuncerà "Ascolto...". Parla quello che desideri e premi di nuovo la scorciatoia per inviare l'audio.
NVDA + Shift + Z
Invia Contesto della Schermata: Funziona come il comando precedente, ma cattura il testo dove si trova il cursore di NVDA e lo allega alla tua domanda.
NVDA + Alt + Z
Cancella Cronologia: Ripristina la memoria del contesto della conversazione corrente. Questo comando non cancella la cronologia salvata in un file.

3. Configurazione dei Provider IA

Vai a NVDA + N > Preferenze > Impostazioni > NVoice per configurare:

Groq (Consigliato per STT)

Groq utilizza un'architettura di elaborazione ultraveloce (LPU). Consigliato per la trascrizione vocale (STT) usando Whisper-Large.

  • Chiave API: Groq Console
  • Costo: Quota gratuita estremamente generosa.

Google Gemini

Intelligenza artificiale di Google. Eccellente per risposte fluenti e analisi dettagliate.

  • Chiave API: Google AI Studio

OpenAI (ChatGPT)

Offre i potenti modelli GPT-5 e superiori.

  • Chiave API: OpenAI Platform

OpenRouter (Multi-Modelli)

Con una sola chiave, accedi a centinaia di modelli (Claude, Mistral, Llama, ecc.).

  • Chiave API: OpenRouter Keys

4. Privacy Totale: IA Locale con Ollama

Per privacy assoluta o lavoro offline, NVoice si integra con Ollama. Configura all'indirizzo predefinito http://localhost:11434.

Modelli Consigliati per Computer Modesti:

  • qwen2:0.5b - Estremamente piccolo e veloce.
  • llama3.2:1b - Super leggero, eccellente per testi rapidi.
  • phi3:mini - Alta capacità di ragionamento.

Per scaricare: ollama pull llama3.2:1b

5. Tutorial: Installare Whisper e Ollama (Windows)

Passaggio 1: Installare Python e FFmpeg

Apri il Prompt dei Comandi come Amministratore ed esegui:

winget install Python.Python.3.11

winget install Gyan.FFmpeg

Riavvia il computer dopo aver installato FFmpeg.

Passaggio 2: Installare Whisper

pip install -U openai-whisper

Passaggio 3: Installare Ollama

winget install Ollama.Ollama

Passaggio 4: Scaricare i Modelli

ollama pull qwen2.5:0.5b

ollama pull llama3.2:1b
