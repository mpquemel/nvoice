NVoice - مساعد الصوت الذكي لـ NVDA

المؤلف: ملكيم كيميل (Melhym Quemel)

الإصدار: 2.0.0

1. حول NVoice

NVoice هو إضافة تحول NVDA إلى مساعد محادثة ذكي منخفض التأخير جداً. باستخدامه، يمكنك المحادثة مع نماذج الذكاء الاصطناعي الأكثر تقدماً في العالم باستخدام صوتك فقط. لا يستخدم NVoice مكتبات خارجية ثقيلة؛ يقوم بتدفق نص الذكاء الاصطناعي مباشرة إلى صوت NVDA الأصلي في الوقت الفعلي، وهو متوافق مع NVDA 32 بت و64 بت.

2. اختصارات لوحة المفاتيح (الأوامر العامة)

تم تصميم NVoice ليتم تشغيله بسرعة دون مقاطعة سير عملك:

NVDA + Z
بدء/إيقاف المحادثة: اضغط ليبدأ NVoice في الاستماع إلى ميكروفونك. سيعلن NVDA "جاري الاستماع...". تحدث ما تريد واضغط الاختصار مرة أخرى لإرسال الصوت.
NVDA + Shift + Z
إرسال سياق الشاشة: يعمل مثل الأمر السابق، لكنه يلتقط النص حيث يكون مؤشر NVDA مركزاً ويلحقه بسؤالك.
NVDA + Alt + Z
مسح السجل: يعيد تعيين ذاكرة السياق للمحادثة الحالية. هذا الأمر لا يحذف السجل المحفوظ في الملف.

3. تكوين مزودي الذكاء الاصطناعي

انتقل إلى NVDA + N > التفضيلات > الإعدادات > NVoice للتكوين:

Groq (موصى به لـ STT)

يستخدم Groq بنية معالجة فائقة السرعة (LPU). موصى به لتحويل الصوت إلى نص (STT) باستخدام Whisper-Large.

  • مفتاح API: Groq Console
  • التكلفة: حصة مجانية سخية جداً.

Google Gemini

ذكاء اصطناعي من جوجل. ممتاز للاستجابات السلسة والتحليلات التفصيلية.

  • مفتاح API: Google AI Studio

OpenAI (ChatGPT)

يقدم نماذج GPT-5 القوية وما فوقها.

  • مفتاح API: OpenAI Platform

OpenRouter (نماذج متعددة)

بمفتاح واحد، يمكنك الوصول إلى مئات النماذج (Claude, Mistral, Llama, إلخ).

  • مفتاح API: OpenRouter Keys

4. الخصوصية الكاملة: تشغيل الذكاء الاصطناعي محلياً مع Ollama

للخصوصية المطلقة أو العمل دون اتصال، يتكامل NVoice مع Ollama. اضبط على العنوان الافتراضي http://localhost:11434.

النماذج الموصى بها للحواسيب المتواضعة:

  • qwen2:0.5b - صغير وسريع جداً.
  • llama3.2:1b - خفيف جداً، ممتاز للنصوص السريعة.
  • phi3:mini - قدرة عالية على التفكير.

للتنزيل: ollama pull llama3.2:1b

5. tutorial: تثبيت Whisper و Ollama (Windows)

الخطوة 1: تثبيت Python و FFmpeg

افتح موجه الأوامر كمسؤول ونفذ:

winget install Python.Python.3.11

winget install Gyan.FFmpeg

أعد تشغيل الكمبيوتر بعد تثبيت FFmpeg.

الخطوة 2: تثبيت Whisper

pip install -U openai-whisper

الخطوة 3: تثبيت Ollama

winget install Ollama.Ollama

الخطوة 4: تنزيل النماذج

ollama pull qwen2.5:0.5b

ollama pull llama3.2:1b

تم التطوير بدقة تقنية وإمكانية وصول حقيقية بواسطة Melhym Quemel.
