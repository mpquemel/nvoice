NVoice - Asistente de Voz con IA para NVDA

Autor: Melhym Quemel

Versión: 2.0.0

1. Acerca de NVoice

NVoice es una extensión que transforma NVDA en un asistente conversacional de IA con latencia muy baja. Con él, puedes dialoglar con los modelos de IA más avanzados del mundo usando solo tu voz. NVoice no utiliza bibliotecas externas pesadas; transmite el texto de IA directamente a la voz nativa de NVDA en tiempo real, compatible con NVDA de 32 bits y 64 bits.

2. Atajos de Teclado (Comandos Globales)

NVoice está diseñado para activarse rápidamente sin interrumpir tu flujo de trabajo:

NVDA + Z
Iniciar/Detener Conversación: Presiona para que NVoice comience a escuchar tu micrófono. NVDA announce "Escuchando...". Habla lo que desees y presiona el atajo nuevamente para enviar el audio.
NVDA + Shift + Z
Enviar Contexto de Pantalla: Funciona como el comando anterior, pero captura el texto donde está el cursor de NVDA y lo adjunta a tu pregunta.
NVDA + Alt + Z
Borrar Historial: Restablece la memoria de contexto de la conversación actual. Este comando no borra el historial guardado en archivo.

3. Configurando Proveedores de IA

Ve a NVDA + N > Preferencias > Configuración > NVoice para configurar:

Groq (Recomendado para STT)

Groq utiliza arquitectura de procesamiento ultrarrápida (LPU). Recomendado para transcripción de voz (STT) usando Whisper-Large.

  • Clave API: Groq Console
  • Costo: Franja gratuita extremadamente generosa.

Google Gemini

Inteligencia artificial de Google. Excelente para respuestas fluidas y análisis detallados.

  • Clave API: Google AI Studio

OpenAI (ChatGPT)

Ofrece los potentes modelos GPT-5 y superiores.

  • Clave API: OpenAI Platform

OpenRouter (Multi-Modelos)

Con una sola clave, accede a cientos de modelos (Claude, Mistral, Llama, etc.).

  • Clave API: OpenRouter Keys

4. Privacidad Total: IA Local con Ollama

Para privacidad absoluta o trabajo sin conexión, NVoice se integra con Ollama. Configura en la dirección predeterminada http://localhost:11434.

Modelos Recomendados para Computadoras Modestas:

  • qwen2:0.5b - Extremadamente pequeño y rápido.
  • llama3.2:1b - Súper ligero, excelente para textos rápidos.
  • phi3:mini - Alta capacidad de razonamiento.

Para descargar: ollama pull llama3.2:1b

5. Tutorial: Instalar Whisper y Ollama (Windows)

Paso 1: Instalar Python y FFmpeg

Abre el símbolo del sistema como Administrador y ejecuta:

winget install Python.Python.3.11

winget install Gyan.FFmpeg

Reinicia tu computadora después de instalar FFmpeg.

Paso 2: Instalar Whisper

pip install -U openai-whisper

Paso 3: Instalar Ollama

winget install Ollama.Ollama

Paso 4: Descargar Modelos

ollama pull qwen2.5:0.5b

ollama pull llama3.2:1b
